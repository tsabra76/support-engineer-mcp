# Knowledge Base

This folder contains the knowledge base for the Support Engineer MCP Server. Replace the sample files with your own data.

## Files

- `tickets_sample.csv` – Sample support ticket data.
- `articles_sample.csv` – Sample knowledge articles.
- `troubleshooting_guide.md` – Sample troubleshooting guide.

You can add any CSV, Markdown, or text files here. The server will recursively search this folder for content.

To use your own data, replace these files with your own and ensure the column names match the expected format (or update the server code accordingly).
